# Navbar

> A simple, customizable navigation bar.

## Usage

```tsx

function App() {
  return (
    <Navbar
      bgColor="#000"
      linkColor="#fff"
      data={['Home', 'About', 'Services', 'Contact']}
      brand={{ text: 'Surya', href: '/' }}
    />
  );
}
